Browser Window Assistant - 隐私政策

最后更新： 2025年9月6日

1. 概述

Browser Window Assistant（"浏览器窗口助手"）致力于保护用户隐私。本隐私政策说明我们如何处理您的信息。

2. 信息收集

我们的Chrome扩展承诺：

  • 不收集个人信息 - 我们不收集、存储或传输任何个人身份信息
  • 本地存储优先 - 所有数据仅存储在您的本地设备上
  • 无服务器传输 - 不向任何外部服务器发送您的浏览数据
  • 无跟踪技术 - 不使用任何跟踪、分析或广告技术

3. 数据处理

扩展仅在本地处理以下信息：

  • 标签页的URL和标题
  • 标签页分组信息
  • 网页的公开描述信息（meta标签）
  • 保存时间戳

重要：所有这些信息仅用于实现扩展的核心功能（保存和恢复浏览会话），永远不会离开您的设备。

4. 权限说明

扩展请求的权限及其用途：

  • tabs - 获取当前标签页信息用于会话保存
  • storage - 在本地存储会话数据
  • tabGroups - 支持标签页分组功能
  • scripting - 提取网页公开信息以改善用户体验
  • history - 可选的历史快照生成功能
  • notifications - 显示操作结果通知

5. 数据安全

  • 所有数据使用Chrome的安全存储API加密保存
  • 不与第三方共享任何信息
  • 您可以随时通过Chrome设置清除所有存储的数据
  • 卸载扩展会自动删除所有相关数据

6. 用户控制

您拥有完全的数据控制权：

  • 随时查看、修改或删除保存的会话数据
  • 选择使用或不使用特定功能
  • 随时卸载扩展并删除所有数据

7. 更新通知

如果此隐私政策发生重大更改，我们会通过扩展更新或GitHub仓库通知用户。

如有隐私相关问题，请通过以下方式联系：

  • GitHub Issues: 项目地址
  • 邮箱: [您的邮箱地址]

Browser Window Assistant - Privacy Policy

Last Updated: September 6, 2025

1. Overview

Browser Window Assistant is committed to protecting user privacy. This privacy policy explains how we handle your information.

2. Information Collection

Our Chrome extension promises:

  • No Personal Data Collection - We do not collect, store, or transmit any personally identifiable information
  • Local Storage Only - All data is stored only on your local device
  • No Server Transmission - We do not send your browsing data to any external servers
  • No Tracking - We do not use any tracking, analytics, or advertising technologies

3. Data Processing

The extension only processes the following information locally:

  • Tab URLs and titles
  • Tab group information
  • Public page descriptions (meta tags)
  • Save timestamps

Important: All this information is used solely to implement the extension's core functionality (saving and restoring browsing sessions) and never leaves your device.

4. Permission Explanations

The extension's requested permissions and their purposes:

  • tabs - Access current tab information for session saving
  • storage - Store session data locally
  • tabGroups - Support tab grouping functionality
  • scripting - Extract public page information to enhance user experience
  • history - Optional historical snapshot generation feature
  • notifications - Display operation result notifications

5. Data Security

  • All data is encrypted and saved using Chrome's secure storage APIs
  • No information is shared with third parties
  • You can clear all stored data at any time through Chrome settings
  • Uninstalling the extension automatically deletes all related data

6. User Control

You have complete control over your data:

  • View, modify, or delete saved session data at any time
  • Choose to use or not use specific features
  • Uninstall the extension and delete all data at any time

7. Update Notifications

If there are significant changes to this privacy policy, we will notify users through extension updates or the GitHub repository.
